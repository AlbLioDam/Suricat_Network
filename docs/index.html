Suricat

This project is generated with yo angular generator version 0.15.1.

Build & development

Run grunt for building and grunt serve for preview.

Testing

Running karma start will run the unit tests with karma and mochai with chai.

Docs

Angular docs are generated with Gulp and JSDocs Angular docs are availables in /docs directory If you need regenerate docs, run $ gulp docs in your command line

Back-end

Don't forget this angular application need the REST server Le Cerbere for back-end

Running

to launch Suricat app, go to \Suricat_Network\app\index.html
